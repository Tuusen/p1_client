## skill_config.json 技能配置

### skill 表各字段含义

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 技能唯一标识符，相同 id 的不同 level 表示同一技能的不同等级 |
| `level` | int | 技能等级，0 级为初始状态（仅用于显示），1-10 级为可用等级 |
| `name` | string | 技能名称 |
| `desList` | string[] | 技能描述列表，每行一个描述，支持升级效果说明 |
| `icon` | string | 技能图标路径（在资源文件夹中的相对路径） |
| `category` | string | 技能类型枚举值：<br>- `Projectile`: 投射物类技能<br>- `Self`: 自身增益类技能<br>- `Shield`: 护盾类技能<br>- `Aoe`: 范围攻击技能<br>- `Summon`: 召唤类技能 |
| `dragHint` | string | 拖拽释放提示文本（可选） |
| `dmg` | int | 技能基础伤害值 |
| `dmgType` | int | 伤害类型：0=无属性, 1=火, 2=冰, 3=电, 4=风 |
| `mp` | int | 消耗的魔法值/能量值 |
| `mpType` | int | 消耗类型：0=无消耗, 1=火能, 2=冰能, 3=电能, 4=风能 |
| `bulletSpeed` | float | 子弹飞行速度（0 表示不使用子弹） |
| `atkCnt` | int | 一次攻击发射的子弹数量 |
| `cd` | float | 技能冷却时间（秒），从技能配置中读取 |
| `bulletStyleId` | int | 子弹样式 ID，关联 bullet_config.json |
| `attack_range` | float | 攻击范围（射程），从技能配置中读取， Projectile 类型技能必需 |
| `events` | object[] | 技能事件列表，包含特殊效果和附加能力 |

### event 事件类型详解

| type | 事件名称 | param 参数说明 | 效果描述 |
|------|---------|---------------|---------|
| 1 | 穿透 | `[穿透数量]` | 子弹可以穿透多个敌人，穿透数量为 param[0] |
| 2 | 爆炸 | `[爆炸伤害, 爆炸半径]` | 子弹命中后爆炸，造成范围伤害 |
| 3 | 冰冻 | `[冰冻时间(秒)]` | 对敌人造成冰冻效果，持续指定时间 |
| 4 | 灼烧 | `[每秒伤害, 持续时间(秒)]` | 对敌人施加持续灼烧伤害 |
| 5 | 多重射击 | `[额外子弹数, 分散角度]` | 发射时额外产生多个子弹，分散角度决定扇形范围 |
| 6 | 闪电连锁 | `[弹射次数, 范围半径, 最小距离, 初始伤害修正]` | 命中后向周围敌人弹射，每次弹射伤害可衰减 |
| 7 | 减速 | `[减速百分比, 持续时间(毫秒)]` | 对敌人施加减速效果，param[0] 为减速比例 |
| 8 | 治疗 | `[治疗量]` | 立即恢复指定数值的生命值 |
| 9 | 持续治疗 | `[每秒治疗量, 持续时间(秒)]` | 持续恢复生命值效果 |
| 10 | 减伤 | `[减伤百分比, 持续时间(毫秒)]` | 提升自身减伤能力，持续指定时间 |
| 11 | 扣血 | `[扣血量]` | 消耗自身当前生命值 |
| 12 | 获得经验 | `[经验值, 技能数量(-1=全部)]` | 为技能获得经验值，可指定随机技能数量 |
| 13 | 护盾 | `[护盾值]` | 为自身添加护盾，可以吸收伤害 |
| 14 | 反击 | `[伤害百分比, 反弹次数, 穿透数量]` | 护盾被攻击时反弹子弹造成伤害 |
| 15 | 击退 | `[击退力度]` | 将敌人向后击退指定距离 |
| 16 | 易伤 | `[易伤百分比, 持续时间(毫秒)]` | 使敌人受到的伤害增加 |
| 17 | 召唤 | `[持续时间, 继承属性百分比, 额外召唤数, 怪物ID]` | 召唤友军单位协助战斗 |
| 18 | 追踪弹 | `[]` | 子弹具有自动追踪敌人的能力 |
| 19 | 护盾破裂 | `[击退力度, 恢复生命值百分比]` | 护盾破裂时触发击退和恢复效果 |


## hero_config.json 英雄配置

### 英雄属性字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 英雄唯一标识符 |
| `name` | string | 英雄名称 |
| `description` | string | 英雄描述，说明英雄特点和定位 |
| `role` | int | 角色形象 ID |
| `attack_skill_ids` | int[] | 普攻技能 ID 数组，关联 skill_config.json，每个技能的 cd 从技能配置中读取 |
| `skill_xp_interval` | float | 获得技能经验的时间间隔（秒） |
| `skill_xp_min` | int | 每次获得的最小技能经验值 |
| `skill_xp_max` | int | 每次获得的最大技能经验值 |
| `attrs` | object[] | 英雄属性列表，每个属性包含 id 和 value |

### attrs 数组中各 id 对应的属性类型

详见 [attribute_config.json](#attribute_configjson-属性配置)

| id | 属性名称 | 描述 |
|----|---------|------|
| 1 | 生命值 | 最大生命值 |
| 2 | 护盾值 | 最大护盾值 |
| 3 | 攻击力 | 基础攻击力 |
| 4 | 攻击范围 | 攻击射程 |
| 5 | 攻击间隔 | 攻击冷却秒数 |
| 6 | 移动速度 | 移动速度 |
| 7 | 伤害值 | 碰撞/攻击伤害 |
| 8 | 攻击数量 | 普攻同时攻击的目标数量 |

## arcane_config.json 奥术配置

### 奥术配置字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 奥术唯一标识符（4001-4009） |
| `name` | string | 奥术名称 |
| `desList` | string[] | 奥术描述列表 |
| `icon` | string | 奥术图标路径 |
| `dmg` | int | 奥术基础伤害值 |
| `dmgType` | int | 伤害类型：1=火, 2=冰, 3=电, 4=风 |
| `radius` | float | 作用半径（-1 表示全屏） |
| `tickInterval` | float | 持续伤害的时间间隔（秒） |
| `cd` | float | 冷却时间（秒） |
| `runeCost` | int | 消耗的符文数量 |
| `runeType` | int | 符文类型：1=火符文, 2=冰符文, 3=电符文, 4=风符文 |
| `events` | object[] | 奥术事件列表（与技能事件系统相同） |

## monster_config.json 怪物配置

### 怪物配置字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 怪物唯一标识符<br>- 1-99: 普通怪物<br>- 100-999: Boss怪物<br>- 1000+: 召唤物/友军 |
| `name` | string | 怪物名称 |
| `role` | int | 角色类型（同英雄配置） |
| `level` | int | 怪物等级 |
| `is_boss` | boolean | 是否为Boss |
| `is_elite` | boolean | 是否为精英怪 |
| `attack_skill_ids` | int[] | 攻击技能 ID 数组，关联 skill_config.json，每个技能的 cd 从技能配置中读取 |
| `attack_interval` | float | 攻击间隔（秒），仅当没有攻击技能时使用 |
| `attrs` | object[] | 怪物属性列表 |

## level_config.json 关卡配置

### 关卡配置字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 关卡唯一标识符 |
| `name` | string | 关卡名称 |
| `des` | string | 关卡描述 |
| `bg` | string | 背景资源路径 |
| `conditions` | int[] | 解锁条件（前置关卡ID列表） |
| `hard` | int | 关卡难度值 |
| `spawn_interval` | float | 怪物生成间隔（秒） |
| `coinNormalKill` | int | 击杀普通怪物获得的金币 |
| `coinEliteKill` | int | 击杀精英怪物获得的金币 |
| `coinBossKill` | int | 击杀Boss获得的金币 |
| `coinSelfDestructRate` | float | 怪物自毁时的金币掉落率 |
| `monsterList` | object[] | 普通怪物生成列表 |
| `superMList` | object[] | 精英怪物生成列表 |
| `bossList` | object[] | Boss生成列表 |
#### 子字段说明
**monsterList / superMList / bossList:**
| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 怪物ID |
| `generate` | int | 生成权重（数字越大生成概率越高） |
| `num` | int | 需要击杀的数量（仅bossList） |


## attribute_config.json 属性配置

### 属性类型说明

| id | 属性名称 | 描述 | 适用对象 |
|----|---------|------|---------|
| 1 | 生命值 | 最大生命值 | 所有单位 |
| 2 | 护盾值 | 最大护盾值 | 英雄、部分怪物 |
| 3 | 攻击力 | 基础攻击力 | 所有单位 |
| 4 | 攻击间隔 | 攻击冷却秒数 | 英雄 |
| 5 | 移动速度 | 移动速度 | 怪物、召唤物 |
| 6 | 伤害值 | 碰撞/攻击伤害 | 怪物 |
| 7 | 攻击数量 | 普攻同时攻击的目标数量 | 英雄 |

## 配置文件使用注意事项

### 1. 技能等级系统
- 每个技能有 0-10 级，0 级为初始状态（仅用于显示）
- 同一个技能的不同等级共享相同的 id
- 升级会增强伤害、增加新事件效果

### 2. 伤害类型系统
- 0: 无属性伤害
- 1: 火属性伤害（火能消耗）
- 2: 冰属性伤害（冰能消耗）
- 3: 电属性伤害（电能消耗）
- 4: 风属性伤害（风能消耗）

### 3. 事件参数注意事项
- 多数事件的参数数组索引从 0 开始
- 时间单位：技能cd为秒，事件中的时间有时为毫秒（如事件类型7、10、16）
- 百分比参数通常表示基数的百分比（如 30 表示 30%）

### 4. 关卡解锁系统
- `conditions` 数组为空表示无前置条件
- 关卡按 id 顺序递进
- 关卡难度（hard）影响怪物属性

### 5. 召唤物系统
- 召唤物（如风影守卫、熔岩守卫）的 attrs 列表通常为空
- 属性由召唤技能的 event type 17 动态设置
- 召唤物的 id 从 1000 开始
